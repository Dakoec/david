Front-end

Es la parte de un sitio web que interactúa con los usuarios, por eso decimos que está del lado del cliente.

Back-end

Es la parte que se conecta con la base de datos y el servidor que utiliza dicho sitio web.

Full-stack

Es un término utilizado para describir a los desarrolladores que conocen tanto los lenguajes de frontend como de backend.
